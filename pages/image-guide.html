Image Type When to use Naming Tips Large Medium Small
JPEG Whenever you have many different colors in a picture. All lowercase. example: cat.jpeg
PNG When an image is filled with one color. underscore between words. example: alabama_sunset_med.png
GIF when the image is animated or all one color. Include pixel dimensions of your image. example:baby_yoda_large_480x288
